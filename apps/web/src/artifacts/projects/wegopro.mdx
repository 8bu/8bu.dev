## What I shipped

Four years building B2B corporate travel and expense at scale. Led shipping of recent bookings, policy change indicators, hierarchy approvals, custom fields, the onboarding flow, and the entire back-office dashboard.

## Migration

Led the Nuxt 2 to Nuxt 4 migration using Vue 3 web components as the bridge layer. Migrated the main web app toward a Vue 3 SPA. Solo-migrated the back-office dashboard to Nuxt 4 + Nuxt UI.

## How I worked

- Designed a modular monolith frontend architecture adopted team-wide.
- Owned technical direction and best practices for the frontend team.
- Led code review process and mentored teammates on Vue 3 patterns.
- Worked closely with design and PM to turn specs into shipped features.
- Handled production issues and quick fixes when stuff broke.
