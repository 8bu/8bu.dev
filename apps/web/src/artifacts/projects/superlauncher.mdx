## What it is

SuperLauncher is an on-chain IDO launchpad with staking and a marketplace. I was a core front-end member from day zero, at BlockDevs, and helped take it from an empty repo to a live platform.

## What I shipped

- Built the front-end architecture from scratch and set the conventions the rest of the team followed.
- A custom Vue 3 wallet module - my own abstraction over WalletConnect, Web3.js, and Ethers.js, integrating MetaMask, TrustWallet, Safepal, Solflare, and Onto.
- IDO launch flows, staking, and the marketplace UI.

## Open source

I hit real bugs in WalletConnect while building the wallet module, so I fixed them upstream and contributed the patches back to the project.

## How I worked

- Owned the architecture and code conventions for the front end.
- Held the line on platform stability and UI quality.
- Ran code reviews and kept the bar high.

## Stack details

TypeScript, Vite, Vue 3, UnoCSS, ElementUI, SCSS, Web3.js, Ethers.js, WalletConnect, Vue-i18n. Wallets: MetaMask, TrustWallet, Safepal, Solflare, Onto. Utils: bignumber.js, cleave.js, dayjs, iconify. Design: Figma, AdobeXD.
