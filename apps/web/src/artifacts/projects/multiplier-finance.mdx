## What it is

Multiplier.finance (formerly Pawnhub) is a multi-chain DeFi lending protocol - AAVE-style lending and borrowing across chains. I was the front-end lead during my BlockDevs era, building a live money app where a UI bug is not cosmetic, it is funds.

## What I shipped

- Core lending and borrowing flows.
- Wallet integration across multiple providers and chains.
- Staking.
- The day-to-day UI and the stability that a financial product demands.

## The migration

Led the front end from Vue 2 class-based syntax to the Vue 3 Composition API - cleaner state, stronger types, and code that was far easier to reason about as the protocol grew.

## How I worked

- Built and owned core features end to end.
- Kept a live DeFi app stable, where correctness is non-negotiable.
- Ran code reviews and shared patterns across the team.

## Stack details

Vue 2 → Vue 3, TypeScript, Tailwind / UnoCSS, Antd, Vuetify, SCSS, Web3.js, Ethers.js, WalletConnect, Vue-i18n. Wallets: MetaMask, TrustWallet, Safepal, Solflare, Onto. Utils: bignumber.js, cleave.js, dayjs, iconify. Design: Figma, AdobeXD.
