## What I shipped

Front-end on a multi-chain liquidity protocol (formerly Pawnhub), AAVE-style lending across chains.

## Migration

Led migration from Vue v2.x Class-based syntax to Vue v3.x Composition API.

## How I worked

- Developed core features: lending, wallet integration, staking.
- Maintained platform stability and resolved UI issues.
- Conducted code reviews and facilitated knowledge sharing among team members.

## Stack details

Vue v2.x, Vue v3.x, Antd, Vuetify, TailwindCSS / UnoCSS, ESLint, SCSS, Figma, AdobeXD, WalletConnect, Web3.js, Ethers.js, Vue-i18n. Wallet integration: MetaMask, TrustWallet, Safepal, Solflare, Onto. Utilities: bignumber.js, cleave.js, dayjs, iconify.
