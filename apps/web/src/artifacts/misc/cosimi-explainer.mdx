## What this is

A SimSimi-style pattern-matching chatbot. No LLM in the request path. Replies come from a curated pair store scored by a three-tier engine: exact match -> Postgres FTS -> trigram similarity.

## Why

If I do not have an answer to your question, I would rather say so than invent one. The honesty trade is worth the coverage gap.

## Where the code lives

Cosimi is open. See [the write-up on Substack](https://0x8bu.substack.com/p/the-r-in-rag-hold-the-g) for the deeper architecture.
