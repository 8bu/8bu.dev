## Editor and AI

Claude Code, Codex, Cursor, OpenCode. Day-to-day driving switches between them depending on the task shape. Claude Code for autonomous work, Cursor for tight in-editor iteration.

## Terminal

Ghostty + FishShell. Lazygit for almost everything in the git graph.

## Local models

Ollama and LMStudio when I want to test something offline or compare against the hosted models.

## Notes and ops

Obsidian for personal notes. Notion / Linear / Jira / Figma / Storybook depending on which team is asking. n8n for personal automation.

## What I do not use

No IDE wizards. No autocomplete-only setups. The AI either drives the loop or stays out of it.
