SuperLauncher is an IDO, staking, and marketplace blockchain platform. An IDO is an Initial DEX Offering - SuperLauncher was an IDO launchpad. I was a core member and fullstack web developer from day zero, at BlockDevs, with the project running Feb 2020 to Jan 2022.

## What I shipped

Core member on a blockchain platform spanning IDO launches, staking, and a marketplace. I built the platform architecture from scratch - Node.js/Express backend and web app - with REST APIs, auth and sessions, PostgreSQL schema and queries, and The Graph subgraphs with GraphQL for on-chain data.

## How I worked

I built the platform architecture from scratch, developed a custom blockchain wallet module for Vue v3.x, contributed bug fixes to the WalletConnect open-source project, maintained platform stability and resolved UI issues, conducted code reviews, and maintained code quality standards.

## The wallet module

I built a custom blockchain wallet module for Vue 3. It integrated MetaMask, TrustWallet, Safepal, Solflare, and Onto via WalletConnect, Web3.js, and Ethers.js.

## Stack

Node.js, Express, PostgreSQL, GraphQL, The Graph (subgraphs), TypeScript, Vite, Vue v3.x, UnoCSS, Figma, AdobeXD, WalletConnect, Web3.js, Ethers.js, Vue-i18n, ElementUI. Wallet integration: MetaMask, TrustWallet, Safepal, Solflare, Onto. Utilities: bignumber.js, cleave.js, dayjs, iconify, ESLint.
