Multiplier.finance is a multi-chain DeFi liquidity protocol - AAVE-style lending across chains. It was formerly called PawnHub. I was Lead Web Developer on it at BlockDevs, 2019-2021, owning everything outside the smart contracts: backend, web app, and the integration between them.

## What I shipped

A multi-chain liquidity protocol, AAVE-style lending across chains. I developed the core features - lending, wallet integration, and staking - and built the Express APIs, auth and sessions, and the PostgreSQL schema and queries behind them, plus The Graph subgraphs and GraphQL queries for on-chain indexing.

## The migration

I led the migration from Vue v2.x class-based syntax to Vue v3.x Composition API.

## How I worked

I developed core features, maintained platform stability and resolved UI issues, and conducted code reviews while facilitating knowledge sharing among the team.

## Stack

Node.js, Express, PostgreSQL, GraphQL, The Graph (subgraphs), Vue v2.x, Vue v3.x, Antd, Vuetify, TailwindCSS / UnoCSS, ESLint, SCSS, Figma, AdobeXD, WalletConnect, Web3.js, Ethers.js, Vue-i18n. Wallet integration: MetaMask, TrustWallet, Safepal, Solflare, Onto. Utilities: bignumber.js, cleave.js, dayjs, iconify.
